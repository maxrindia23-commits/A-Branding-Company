# Adzangaadi Website Clone

A pixel-close recreation of [adzangaadi.com](https://adzangaadi.com) built with **React**, **Tailwind CSS v4**, **Framer Motion**, and **Lenis** smooth scrolling.

## Features

- Full landing page section flow matching the original site
- Dark theme with neon lime accents and Clash Display typography
- Cinematic Lenis smooth scroll
- Scroll-triggered fade/slide animations (Framer Motion)
- Responsive navbar with mobile drawer
- Reusable UI components (buttons, pill labels, fade-in wrappers)

## Getting started

```bash
cd adzangaadi-clone
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build

```bash
npm run build
npm run preview
```

## Stack

- Vite + React 19
- Tailwind CSS 4 (`@tailwindcss/vite`)
- Framer Motion 11
- Lenis 1.x

## Note

Images and video are loaded from the live adzangaadi.com CDN for visual fidelity. Replace with local assets for production if needed.
